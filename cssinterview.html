what is css?why do we use it?

CSS (Cascading Style Sheets) is a style sheet language used to control the visual presentation of web pages. It describes how HTML elements should be displayed on screen, paper, or other media.

What is specificity among the ways of css styling?

**CSS specificity** is a ranking system used by browsers to determine which styles to apply when multiple styles target the same element. Itâ€™s calculated based on the type of CSS selector used:
- **Inline styles** have the highest specificity.
- **ID selectors** (`#id`) are stronger than class selectors or element selectors.
- **Class selectors** (`.class`) are stronger than element selectors (`p`, `h1`).
- **Element selectors** have the lowest specificity.
When styles conflict, the rule with the highest specificity is applied.

What is the syntax of css?

The syntax of CSS consists of a selector and a declaration block.
Selector: Specifies which HTML element(s) to style.
Declaration Block: Contains one or more declarations enclosed in curly braces {}. Each declaration includes:
Property: The CSS attribute you want to style (e.g., color, font-size).
Value: The value for the property (e.g., blue, 16px).

What is a selector?

A CSS selector is a pattern used to target and select the HTML elements you want to style. It allows you to apply specific styles to elements based on their tag, class, ID, or other attributes

How many types of selectors?

There are five main types of CSS selectors:
1. **Universal Selector (`*`)**: Selects all elements.
2. **Type Selector (Element Selector)**: Selects elements by tag name (e.g., `h1`, `p`).
3. **Class Selector (`.className`)**: Selects elements with a specific class attribute.
4. **ID Selector (`#idName`)**: Selects an element with a specific ID.
5. **Attribute Selector**: Selects elements based on attributes (e.g., `[type="text"]`).
Each type targets different elements to apply styles efficiently.

What is d/f b/w the pseudo class vs pseudo element?

A pseudo-class targets an element based on its state (e.g., :hover, :focus, :nth-child), whereas a pseudo-element targets a specific part of an element's content (e.g., ::before, ::after, ::first-letter).

What is rule set?

A rule set in CSS is a block of code that defines the styling for one or more HTML elements. It consists of a selector and a declaration block.

What is box-modal?

Box model is a fundamental concept in CSS that describes the rectangular boxes generated for elements in a web page layout. Each box consists of four components:
Content: The actual content of the element (text, images, etc.).
Padding: The space between the content and the border, which adds space inside the box.
Border: The line surrounding the padding (if any) and content.
Margin: The space outside the border, which separates the element from other elements

What is the difference between border and outLine?

Border: A border is a line that wraps around an element's content and padding. It affects the layout by contributing to the element's size and can be styled with different widths, colors, and styles (solid, dashed, etc.).
Outline: An outline is a line that surrounds an element but does not take up space or affect the layout. It is drawn outside the border and can also be styled with different widths and colors

Types of colors we have in css?

In CSS, there are several types of colors you can use:
Named Colors: Predefined color names (e.g., red, blue, green).
Hexadecimal Colors: A six-digit code representing colors (e.g., #ff5733).
RGB Colors: Defines colors using the red, green, and blue components (e.g., rgb(255, 87, 51)).

HSLA Colors: Similar to HSL but includes an alpha value for transparency (e.g., hsla(12, 100%, 60%, 0.5)).

What is opacity?

**Opacity** is a CSS property that determines the transparency level of an element. It ranges from `0` (completely transparent) to `1` (completely opaque). For example, setting `opacity: 0.5` makes the element semi-transparent, allowing the background to show through. It can be useful for layering elements and creating visual effects.

What are shadows in css and how many are they?

In CSS, shadows refer to the visual effects that create the illusion of depth by adding shadows to elements. The two main types of shadows are:
**Box Shadow** (`box-shadow`): Adds a shadow effect around an element's box (like a div or button). It can specify horizontal and vertical offsets, blur radius, spread radius, and color.
```css box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5);
``` **Text Shadow** (`text-shadow`): Applies a shadow effect to text, enhancing its visibility and style. It also accepts offsets, blur radius, and color.
```css text-shadow: 1px 1px 3px rgba(0, 0, 0, 0.7);
``` These shadows enhance the visual aesthetics of web elements, making them stand out and improving the user experience.

What are gradients in css and how many are they?

Gradients in CSS are smooth transitions between two or more colors. They can be used as backgrounds or for other styling effects.
Types of CSS Gradients:
Linear Gradients: Transition along a straight line. Defined with the linear-gradient() function.
Example: background: linear-gradient(to right, red, yellow);
Radial Gradients: Transition outward from a central point. Defined with the radial-gradient() function.
Example: background: radial-gradient(circle, red, yellow);
Conic Gradients: Transition around a central point, creating a pie-like effect. Defined with the conic-gradient() function.

What is the d/f b/w display and float?

What is d/f b/w the display:inline vs display:block vs display:inline-block?

display: block: Elements take up the full width available and start on a new line. Examples include div and p. They create a "block" structure.
display: inline: Elements only take up as much width as necessary and do not start on a new line. Examples include span and a. They flow within the content, allowing other elements to sit next to them.
display: inline-block: Combines characteristics of both. Elements flow inline like inline elements but can have width and height set, and margins/padding applied. This is useful for creating layouts without breaking the flow.

visibility: hidden vs. display: none:

what is object-fit ? and in how many types can we fit the object?

object-fit is a CSS property that defines how an element, typically an img or video, should be resized to fit its container. It controls the content's sizing and alignment, ensuring it displays correctly within the given dimensions.
Types of object-fit Values: fill: The default value. The content stretches to fill the container, which may distort the aspect ratio.
contain: The content scales to fit within the container while maintaining its aspect ratio. The entire object is made to fit, possibly leaving empty space.
cover: The content scales to cover the entire container while preserving its aspect ratio. Parts of the content may be clipped to fill the space.
none: The content retains its original size, which may result in overflow if it exceeds the container's dimensions.
scale-down: The content is sized as none or contain, whichever results in a smaller size.

What is background property? what are the properties of background do you know?

The background property in CSS is a shorthand property used to set multiple background-related properties for an element in a single declaration. It allows you to control the background color, image, position, size, repeat, and attachment.
Key Properties of the Background:
background-color: Sets the background color of an element.
background-image: Specifies an image to be used as the background.
background-repeat: Defines how the background image is repeated (e.g., no-repeat, repeat, repeat-x, repeat-y).
background-position: Sets the starting position of the background image (e.g., center, top right, etc.).
background-size: Specifies the size of the background image (e.g., cover, contain, or specific dimensions).
background-attachment: Controls whether the background image scrolls with the page or is fixed (e.g., scroll, fixed, local).

what is the position property?

The position property in CSS determines how an element is positioned in the document flow. It has several values:
static: Default value; elements are positioned according to the normal flow of the document.
relative: Positioned relative to its normal position; allows for offsets using top, right, bottom, and left.
absolute: Positioned relative to the nearest positioned ancestor (not static); removed from the normal flow.
fixed: Positioned relative to the viewport; remains in the same place even when the page is scrolled.
fixed: Toggles between relative and fixed, depending on the scroll position.

What is overflow in css?

Overflowing in CSS refers to the situation when an element's content exceeds its specified width or height. This can result in content being cut off or overflowing outside the container.
CSS Overflow Properties:
visible: Content is not clipped and will overflow the element.
hidden: Content is clipped, and the overflow is not visible.
scroll: Content is clipped, but scrollbars are added to view the overflow.
auto: Scrollbars appear only if the content overflows.

What is a clear property?

The clear property in CSS is used to control the behavior of floating elements. It specifies whether an element should be moved below (cleared) any preceding floated elements.
Values:
left: The element is moved below any left-floated elements.
right: The element is moved below any right-floated elements.
both: The element is moved below all floated elements.
none: The default; the element can sit next to floated elements.

what is d.f b/w the transitions vs transforms vs animations? 
            In CSS:

### Transitions
- **Definition**: Allow property changes in CSS values to occur smoothly over a specified duration.
- **Usage**: Triggered by state changes (e.g., hover).
- **Example**: Changing the background color when a button is hovered over.

### Transforms
- **Definition**: Modify the coordinate space of an element, allowing for transformations like scaling, rotating, skewing, and translating.
- **Usage**: Applied immediately without a duration.
- **Example**: Rotating an image or moving a box.

### Animations
- **Definition**: Allow for more complex sequences of styles that can change over time, including keyframes for detailed control.
- **Usage**: Can run automatically and loop.
- **Example**: A loading spinner that rotates continuously.

In summary:
- **Transitions**: Smooth changes on state change.
- **Transforms**: Immediate visual changes to an element.
- **Animations**: Complex sequences with keyframes and control over multiple properties.
What is a matrix in transforms?

In CSS transforms, a matrix is a mathematical representation used to define transformations such as translation, rotation, scaling, and skewing of elements. The matrix is a 2D array that combines these transformations into a single function. It allows for complex transformations by manipulating the element's position and size in a coordinate space.
